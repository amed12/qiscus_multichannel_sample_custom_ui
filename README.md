## ğŸ“š Getting Started

### Prerequisites
- Flutter SDK (3.0.0 or higher)
- Dart SDK
- Qiscus Multichannel account

### Installation

1. Clone this repository
```bash
git clone <repository-url>
cd qiscus_multichannel_sample_custom_ui
```

2. Install dependencies
```bash
flutter pub get
```

3. Configure your Qiscus credentials in `lib/core/config/app_config.dart`

4. Run the app
```bash
flutter run
```
